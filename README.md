## Sobre mim

Comecei pela curiosidade no front-end, mas foi no back-end que encontrei meu ritmo. Hoje estou no meu primeiro estágio como desenvolvedor, construindo aplicações com PHP, MySQL e JavaScript — e aprendendo que cada bug resolvido é uma aula que nenhuma faculdade substitui.

Antes disso, passei por um estágio de suporte técnico que me ensinou algo valioso: entender o problema antes de escrever a primeira linha de código.
